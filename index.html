CSS BOX MODEL

What is css box model

This is essentially a box that wraps around every HTML elements

Main properties of css box model

  • Content

    • it is the content of the box where text and images appear
  • Padding

    • it clears an area around the content.The padding is trnsparent
  • Border

    • is a line that goes around the padding and content
  • Margin

    • it clears an area outside the border.The margin is transparent
